## You need AWS CLI Installed on your machine.

```sh
curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"
unzip awscliv2.zip
sudo ./aws/install
```

Verify the installation:

```sh
which aws
aws --version
```

Then configure your AWS credentials:
```sh
aws configure
```
